# Mass Leak Scanner v2.3

Ultra fast mass scanner for finding exposed `.env`, `.git`, `phpinfo`, and other sensitive files.

## Features
- Very fast scanning (supports 90+ threads)
- Strong detection with low false positives
- Clean output (only URLs saved line by line)
- Supports both single target and list of domains
- Beautiful rainbow banner
- Live saving of results

```bash
pip install aiohttp aiofiles colorama
